# nordic-seas-validation

## Quick-start

```shell
git clone git@github.com:JMMP-Group/nordic-seas-validation.git
cd nordic-seas-validation
conda env create -f environment.yml
conda activate nordic-seas-validation
pip install -e .
python -c "import src"
```
